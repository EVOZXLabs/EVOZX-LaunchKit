🚀 Deploy A Token

1️⃣ Connect Wallet

TokenPocket, OKX Wallet, Bitget Wallet, Rabby, or MetaMask.

2️⃣ Switch To EVOZ Mainnet

Automatic network detection and switching.

3️⃣ Configure Token

- Name
- Symbol
- Supply

4️⃣ Deploy

Approve the transaction.

5️⃣ Done

Your token is now live on EVOZ Mainnet.
